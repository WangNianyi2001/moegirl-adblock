# 萌娘百科广告屏蔽器

屏蔽萌娘百科（`https://zh.moegirl.org.cn/*`）上的广告，包括 AdBlocker 检测弹窗。

> For those who don't know, `moegirl.org.cn` is NOT a pedophilia website.
> It is a Chinese online encyclopedia about animes, games, memes, etc.
> I forgive you for not knowing Chinese Internet :))

安装地址：https://github.com/WangNianyi2001/moegirl-adblock/raw/refs/heads/master/moegirl-adblock.user.js

## 注意事项

- 宿主于油猴或其他 Userscripts 运行插件。
- 采用了变频轮询的检测策略，广告会闪一下再消失，属于正常现象。
